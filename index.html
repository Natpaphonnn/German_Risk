DATASET

About the Data

German Credit Dataset from a bank in Germany — 1,000 borrower records

🎯

Objective

Classify borrowers as "Good" (low risk) or "Bad" (high risk) based on their financial and personal information.

📊

Class Distribution

70% Good vs 30% Bad — the dataset is imbalanced, so we used SMOTE to handle this during model training.

❓

Missing Values

Saving Account (18.3%) and Checking Account (39.4%) had missing values — filled with "unknown" as a meaningful category.

🔍

Phase 1 — EDA

Explored distributions, correlations, and relationships. Discovered that missing checking account data strongly correlates with high risk.

🛠

Phase 2 — Feature Engineering

Created 3 new features: Monthly Payment (credit/duration), Age Group (Young/Adult/Senior), and Credit per Age. Encoded categoricals with LabelEncoder and scaled with StandardScaler.

🤖

Phase 3 — Modeling

Trained 5 models: Logistic Regression, Random Forest, SVM, Gradient Boosting, XGBoost. Applied SMOTE for class imbalance and GridSearchCV for hyperparameter tuning.

📈

Phase 4 — Evaluation

Evaluated with ROC-AUC, Confusion Matrix, PR Curve, Feature Importance, and SHAP values for model explainability.

INSIGHTS

Key Findings

What the data and model tell us about credit risk

88%
Unknown Checking = Highest Risk
Borrowers with no checking account data have an 88% bad rate — the strongest risk signal in the entire dataset.
58%
Young Borrowers (Age ≤ 25)
The youngest age group has a 58% bad rate, more than double the 25% rate of senior borrowers (age 46+).
Higher
Large Loans = More Risk
The "bad" group has a significantly higher median credit amount. Higher loan amounts correlate with higher default risk.
6-8 mo
Longer Duration = More Risk
Bad borrowers have an average loan duration 6-8 months longer than good borrowers.
